Canonical URL: https://schema.iot.webizing.org/IPFSCamera

schema:Thing > IPCamera > IPFSCamera

IPFSCamera Sensor - video content from the IPFS Camera

Property Expected Type Description
name schema:Text name of the sensor - format: SensorXX ex)Foobot00, Beddit02
user schema:Text user of the sensor - format: firstName+LastName ex)jonghoLee, wanhoIm
address schema:Text building where the sensor resides - format: organization-building ex)yongjaelee-house, kist-l1, kist-l8
room schema:Text room number of the building where the sensor resides - ex) L8321
location schema:Text location of the sensor - format: free
time schema:DateTime time at which the data measurement was made - ex)"2017-05-30T18:54:20+09:00"
url schema:URL link to the video
hash schema:Text hash to the video file in the IPFS system

JSON-LD Usage Example

{
"@context": "https://schema.iot.webizing.org/",
"@type": "IPFSCamera",
"name": "Foobot00",
"user": "jonghoLee",
"address": "kist-l1",
"room": "L8321",
"location": "On the table",
"time": "2017-05-30T18:54:20+09:00",
"url": "www.kist.videos/1"
"hash": "LKlasdfjloewLKDSF1239"
}
